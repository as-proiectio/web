# <img src="public/logo.svg" width="36" height="36" align="center" /> Alpha Signals (알파시그널)

> **우리는 지속가능하고 깨끗한 금융 정보 제공을 추구합니다.**

Alpha Signals는 Core 엔진 시스템을 통해 자동화 발행되는 미국 주식 분석 리포트를 투명하게 아카이빙하고 조회하는 공간입니다.

광고 소음 없이 오염되지 않은 1차 출처 데이터와 AI 기반 매크로 분석 리포트를 왜곡 없이 제공합니다.
